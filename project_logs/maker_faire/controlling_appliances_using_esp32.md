# Controlling Appliances Using ESP32

The ESP32 acts as an access point. The ESP32 hosts a simple webpage that is a control panel for users to be able to manipulate the state of the ESP32 GPIO pins. Two of the GPIO pins are connected to a 2-channel relay module.
